# Lumina — Локальный ИИ-ассистент для работы с файлами 🚀

**Lumina (Lumina Open Web Agent)** — это мощный локальный веб-интерфейс и интеллектуальный агент для управления файлами, структурирования папок и автоматического форматирования текстовых документов. 

Вся обработка, генерация текстов и выполнение файловых операций происходят **абсолютно локально** с помощью ваших собственных LLM через **Ollama**. Никакие конфиденциальные файлы и данные не отправляются во внешние облачные сервисы.

---

## 💡 Задумка и идея проекта

Основная идея Lumina — объединить удобство чат-интерфейса типа ChatGPT с реальной автоматизацией работы над файловой системой вашего компьютера. 

Вместо того чтобы вручную переименовывать десятки файлов, раскладывать их по тематическим папкам или писать контент с нуля, вы просто ставите задачу ассистенту на естественном языке (например: *"Переименуй эти 3 документа, напиши в них рэп про русских классиков в формате .docx и отформатируй их"*). 

Ассистент анализирует содержимое рабочей папки, составляет пошаговый план изменений и выводит его в виде **интерактивных карточек действий**. Вы контролируете весь процесс и можете применить изменения (или отменить их) в один клик.

---

## 🔥 Что крутого в Lumina?

1. **Интерактивные карточки действий (Action Cards)**:
   - ИИ-ответы преобразуются в компактные карточки (создание файлов, переименование, удаление, сортировка, форматирование по ГОСТ).
   - Карточки интерактивны: они отображают только задачу, а технические JSON-параметры приоткрыты частично с эффектом затухания. Клик по плашке раскрывает её полностью.
   - Вы можете выполнить каждое действие отдельно с помощью кнопки **«Принять»** на самой карточке или применить все изменения разом с помощью кнопки **«Подтвердить все»** внизу группы действий.

2. **Полная локальность и приватность**:
   - Приложение работает в связке с локально запущенным API **Ollama**.
   - Доступные модели (Llama 3, Mistral, Gemma, Phi и др.) подтягиваются в интерфейс автоматически.

3. **ChatGPT-style интерфейс**:
   - Современный дизайн с эффектом матового стекла (Glassmorphism), плавной сменой тем и микроанимациями.
   - Удобная овальная строка ввода с кнопками прикрепления файлов и выбора инструментов.
   - **Умный скроллинг (Smart Scroll)**: вы можете свободно читать историю переписки во время генерации ответа — интерфейс не будет насильно прокручивать экран вниз, пока вы сами не вернетесь к нижней границе чата.

4. **Встроенный Word-редактор и ГОСТ-форматирование**:
   - Интегрированный симулятор текстового редактора с поддержкой файлов `.docx`, `.txt` и `.md`.
   - Поддержка стандартов форматирования по ГОСТ (шрифты, размеры, отступы абзацев, межстрочные интервалы) для `.docx` в один клик прямо из панели редактора или через команду ИИ.

---

## 🛠️ Технологический стек

- **Frontend**: Чистый HTML5, современный Vanilla CSS (переменные, Flexbox/Grid, Glassmorphism, CSS-анимации), Vanilla JS (модульный подход, асинхронный стриминг ответов).
- **Backend**: Python 3.12 (HTTP-сервер на `http.server`, интеграция с файловой системой, библиотека `python-docx` для работы с документами Microsoft Word).
- **Контейнеризация**: Docker & Docker Compose для быстрой изоляции и развертывания.

## 📦 Быстрый локальный запуск

Для запуска приложения вам понадобятся установленный **Docker** и запущенный локально **Ollama** (по умолчанию ожидается на `http://localhost:11434`).

### 1. Запуск приложения
Склонируйте репозиторий и выполните скрипт сборки и запуска в корневой папке:

```bash
chmod +x start.sh stop.sh
./start.sh
```

Скрипт автоматически соберет Docker-образ и запустит контейнер `lumina-chat` в фоновом режиме.

### 2. Использование
После успешного запуска откройте браузер и перейдите по адресу:
👉 **[http://localhost:8000](http://localhost:8000)**

1. Введите абсолютный путь к папке на вашем компьютере, в которой вы хотите работать (например, `/home/user/Documents/test`).
2. Выберите модель Ollama из выпадающего списка.
3. Пользуйтесь чатом или встроенными инструментами на панели плюсика `+`.

### 3. Остановка приложения
Для остановки контейнера выполните:

```bash
./stop.sh
```

## ⚙️ Основные поддерживаемые действия ИИ

- `create_file`: Создание нового текстового или `.docx` файла с заполнением контента и возможностью одновременного удаления старого файла (переименование с наполнением).
- `rename`: Быстрое пакетное переименование файлов по регулярным выражениям или паттернам.
- `delete`: Локальное удаление ненужных файлов.
- `sort`: Сортировка файлов по папкам-категориям (документы, изображения, код и т.д.) на основе их расширений.
- `format_docx`: Автоматическое приведение документов в соответствие с ГОСТ.
